1. If ... else

- if block executes only when the condition given in the if is true.
- else block execute when all the if conditions are false, else block has to be at the last position only in the sequence.
- only one block will be executed from all if & else blocks.
- -
Examples -

* Use of 'if' condition to check if given number is less than 7
